# Softmartt

Softmartt is a discovery platform that allows individuals and organizations to create profiles to promote their businesses, services, expertise, charities, or job-seeking activities.

The platform helps customers easily find solutions to their needs by browsing categorized profiles based on location, category, and specialization.

Softmartt acts as a digital marketplace for visibility, connecting service providers and organizations with potential customers.

## Overview

Softmartt enables users to create public profiles describing what they offer.

These profiles are organized into categories and subcategories so customers can easily search and discover relevant providers.

Customers do not need to create accounts to browse profiles.

However, users who want to publish profiles must create an account.

## Categories

Softmartt organizes profiles into five primary categories:

Businesses

Services

Experts

Job Seekers

Charities

Each category contains multiple subcategories.

Example subcategories:

Advertisement

Agriculture

Arts

Automobile

Clothing

Construction

Education

Healthcare

Technology

This structured categorization allows users to publish their profiles in the most relevant sector and helps customers find services quickly.

## Profile System

Users can create multiple profiles under their account.

Each profile represents a specific business, service, organization, or expertise.

A profile includes the following information:

Profile name

Logo

Email address

Physical address

Map location

Category

Subcategory

Description

Social media links

Showcase images

Profiles act as digital storefronts where users present their services or activities.

## Customer Experience

Customers can browse the platform without registering.

They can:

Search profiles by name

Filter by category

Filter by subcategory

Search by location

View detailed profiles

Save and unsave profiles

### Example Use Case

If a user has a car problem, they can:

Open Softmartt

Navigate to Services

Select Automobile

Find mechanics near their location

## Advertisement System

Softmartt includes a paid advertising system that allows users to promote their profiles.

Advertisements appear as image banners displayed in a carousel across the platform.

These ads rotate automatically to give visibility to multiple advertisers.

Clicking an advertisement redirects users to the advertised profile.

## Revenue Model

Softmartt generates revenue through two main channels.

#### Profile Subscriptions

Users may pay subscription fees to maintain or enhance their profiles on the platform.

#### Advertisement Placements

Users can promote their profiles by purchasing advertisement placements.

Advertisements appear as rotating banners within the platform.
